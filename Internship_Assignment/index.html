Project Overview

The mission was to build a fully automated content pipeline. This solution demonstrates a production-ready workflow that is efficient, reliable, and intelligent.

How It Works

The automation triggers when a writer moves a Google Doc into a "Ready to Publish" folder. The n8n workflow then takes over, performing a series of actions: it parses the document to extract the title, body content, tags, and a featured image URL. It then uses the OpenAI API to generate a concise, SEO-friendly meta description. Finally, it publishes the complete article to a Ghost blog, setting the title, content, tags, meta description, and featured imageâ€”all without human intervention. For full reliability, the entire process is wrapped in error handling, with success and failure notifications sent to Slack and a permanent log kept in Google Sheets.

The Development Journey

A successful project is not just about the final result, but the process of overcoming challenges. This section details the strategic decisions and problem-solving that led to the final, robust workflow.

Initial Platform: WordPress.com

I began by building the automation for WordPress.com. While I successfully interacted with its API, I quickly discovered critical limitations in the free tier's media handling capabilities, which made direct file uploads unreliable.

Pivot 1: WordPress.org

Recognizing the platform constraints, I made a strategic pivot to a self-hosted WordPress.org environment. This provided full API access but revealed a deep-seated incompatibility between the n8n version's HTTP node and the specific requirements for multipart file uploads, leading to persistent errors.

Pivot 2 & Final Solution: Ghost

To deliver a flawless and professional solution, I made a final pivot to Ghost. Its modern, well-documented Admin API and first-class n8n integration allowed me to build the final workflow efficiently. This version seamlessly publishes the content, sideloads the featured image from its URL, and sets all metadata in a single, reliable step.
